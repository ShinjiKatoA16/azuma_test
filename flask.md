# Create virtual environment

Open MiniConda(Anaconda) terminal.

- `conda create -n flask python=3.8`
- `conda activate flask`
- `pip install flask`

# 1st step

## Create and move to working directory

- `mkdir Desktop\flask-test`
- `cd Desktop\flask-test`
